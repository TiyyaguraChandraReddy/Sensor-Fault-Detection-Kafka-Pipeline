## STEP 01

```bash
conda create -p ENVIRNONMENT_NAME python=3.8 -y
```

OR

```bash
conda create -n ENVIRNONMENT_NAME python=3.8 -y
```

## STEP 02

```bash
conda activate ENVIRNONMENT_NAME
```

## STEP 03

```bash
pip install -r requirements.txt
```
